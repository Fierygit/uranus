### Uranus

乌拉诺斯（英语：Uranus） [1]  是古希腊神话中的天空之神，从大地之神盖亚的指端诞生，最初作为统治者的众神之王、即天空的神格化。
象征希望与未来，并代表天空。乌拉诺斯既是盖娅的儿子，也是盖娅的丈夫，是十二泰坦神、独眼巨人与百臂巨人的父亲。
